This WooCommerce addon plugin allows you to collect fixed or percentage donations from selected charities based on the shopping cart total.

== Description ==

**WooCommerce Donations Plugin by Circle Fund**
Donations can be shown as regular WooCommerce products or in the shopping cart itself. Options include;

1. Add local charities as non-taxable, non-shippable products.
2. Define donation amount options or assign a percentage donation based on the cart total.
3. If a user skips the shopping cart or goes directly to checkout, a donation link on the checkout page will be shown.
4. Admins can add a donation percentage to apply to the purchases at checkout.
  -e.g. “10% of purchases this month will go to the Red Cross in aid of Hurricane victims.”
5. Admins can set up Campaigns to track donations to difference causes or nonprofits
6. Admins can view reporting on donations by campaign and see which performs best.

== Installation ==

1. Unzip the zipped file and upload to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to your Dashboard / WooCommerce / Donations.
4. Under the *Settings* tab, select a Product that you have created which is non-shippable and non-taxable.
5. Choose how your donation amount should be created; as a custom amount, predefined values or a percentage.
6. If a cart percentage is chosen, select the label and amount here.
7. Under the *Campaign* tab, create a Campaign name as a category for including your donation products.
8. Visit the *Reports* tab to see orders and donations by campaign.

== Frequently Asked Questions ==

= How do I create a Product as a donation? =

Just visit the Dashboard / Products menu, and add a New Product.  Leave all other values empty, such as shipping and tax fields.
